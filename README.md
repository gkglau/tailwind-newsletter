## Newsletter Signup UI

A simple and responsive newsletter signup component built with Tailwind CSS.

This project focuses on creating a clean, modern interface while practicing layout, spacing, and component structure using utility-first CSS.

👉 [View the project live](https://gkglau.github.io/tailwind-newsletter/)

### Features
Responsive layout (mobile-first)
Clean and minimal UI
Email input + call-to-action button
Built with Tailwind CSS
Focus on spacing and typography
### Tech Stack
HTML5
Tailwind CSS
### Purpose

This project is part of my transition from UI/UX Design to Frontend Development.

The goal was to practice:

Translating design into code
Building reusable UI components
Improving visual hierarchy and spacing
### What I learned
How to structure layouts using Tailwind utility classes
Creating consistent spacing without custom CSS
Designing simple but effective call-to-action sections
Improving responsiveness with minimal code
